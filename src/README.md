# 🔷 FiberManager

Sistema de gerenciamento de bobinas de fibra óptica.

## 📋 Funcionalidades

- ✅ Controle de estoque de bobinas
- ✅ Registro de entrada e saída
- ✅ Relatórios por período
- ✅ QR Code para identificação
- ✅ Mapa de localização
- ✅ Modo escuro
- ✅ Responsivo (celular/computador)
- ✅ Auditoria de alterações

## 🎯 Telas do Sistema

1. **Login** - Acesso seguro ao sistema
2. **Dashboard** - Visão geral do estoque
3. **Bobinas** - Cadastro e gerenciamento
4. **Saída** - Registro de consumo
5. **Relatórios** - Análises e exportação
6. **Mapa** - Localização física

## 🛠️ Tecnologias

- HTML5
- CSS3 (TailwindCSS)
- JavaScript
- GitHub Pages
